Abstract presentation layer

The site uses layered gradients, animated signal rings, intentional typography, and sectioned cards so the repo state feels designed instead of dumped.

Reusable scan engine

GitHub scanning now lives in JavaScript modules, not inside workflow YAML. The same scoring logic powers local previews, stored JSON, markdown reports, and issue updates.

Static and dynamic paths

The dashboard reads generated JSON for GitHub Pages, and the local server can also expose a live API route when credentials are available.

Workflow-backed publishing

The scan workflow refreshes report data, uploads artifacts, and can keep the public-facing dashboard current without manual editing.
